# Project Euler #2 - Even Fibonacci Numbers

## Problem

Each new term in the Fibonacci sequence is generated by adding the
previous two terms. Starting with 1 and 2, the sequence begins:

```
1, 2, 3, 5, 8, 13, 21, 34, ...
```

By considering the terms whose values do not exceed **4,000,000**, find
the sum of the even-valued terms.

---

## Solutions

| File          | Description                                                                                 |
| ------------- | ------------------------------------------------------------------------------------------- |
| **short.py**  | Shortest readable Python solution.                                                          |
| **memory.py** | Memory-optimized solution that generates only even Fibonacci numbers (O(1) space).          |
| **visual.py** | Graphically displays the Fibonacci sequence and highlights the even terms using Matplotlib. |

---

## Output

```
4613732
```

---

## Complexity

| File          | Time   | Space |
| ------------- | ------ | ----- |
| **short.py**  | O(n)   | O(1)  |
| **memory.py** | O(n/3) | O(1)  |
| **visual.py** | O(n)   | O(n)  |

---

## Requirements

* Python 3.10+
* Matplotlib (only for **visual.py**)

Install dependencies:

```bash
pip install matplotlib
```

---

## Usage

Run the shortest solution:

```bash
python short.py
```

Run the memory-optimized solution:

```bash
python memory.py
```

Run the visual solution:

```bash
python visual.py
```
